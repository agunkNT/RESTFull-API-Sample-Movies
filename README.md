# RESTFull-API-Sample-Movies
Simple Rest API Movies CRUD in c# .Net Core using MSSQL Database.

ENDPOINTS

Aplication Routes:

  GET - List all movies
  GET/:ID - Detail a single movie
  POST - Create a new movie
  PATCH/:ID - Update a movie
  DELETE - Delete a movie

STRUCTURE

Project Structure:  
      - Models
      - Services
      - Controllers
